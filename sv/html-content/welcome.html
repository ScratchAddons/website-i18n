Scratch Addons är nu installerat!

Tack för att du installerar Scratch Addons. Hoppas du tycker om det!

Getting started

Steg 1
Tryck på Scratch Addons tilläggs ikonen

Den är i det övre högra hörnet i din webbläsare. Google Chrome användare trycker på pusselbiten för att hitta den.

Steg 2
Tryck på kugghjulet och gå till inställnings sidan

Detta kommer att öppna inställnings sidan där du kan konfigurera hur tillägget fungerar.

Steg 3
Bläddra igenom tilläggs listan och sätt på de du vill ha

You can also configure each of the addons on the bottom section of each addons.

Vill du stöta oss?

Du kan stjärna vårt förvar på GitHub eller lämna en recension på Chrome tilläggs sidan, Firefox tilläggs sidan eller Microsoft Edge tilläggs sidan.
