Расширение Scratch Messaging теперь Scratch Addons!

Что произошло?

Расширение Scratch Messaging (от griffpatch) было обновлено и теперь называется Scratch Addons.

Кликните на иконку Scratch Addons в панели расширений чтобы посмотреть новый, переработанное содержание сообщений!

Scratch Addons также включает в себя много других функций по Scratch'у. Для их включения зайдите в настройки Scratch Addons.

Например, Вы можете включить уведомитель Scratch для браузерных уведомлений при получении новых сообщений на Scratch.

Темы и функции для редактора проектов Scratch также доступны, по типу тёмный режим редактора и индикатор загрузки проекта.

Что такое Scratch Addons?

Scratch Addons — браузерное расширение «всё в одном» для Scratch. Оно даёт функции и темы для веб-сайта Scratch и его редактора проектов.

Scratch Addons is completely free and open source. You can read the source code and contribute on GitHub.

Expect weekly or monthly updates, including bug fixes, enhanced features, and new features and themes!

Is there anything that I need to do?

Нет.

Scratch Messaging Extension was already updated to Scratch Addons on your device. You can keep using Scratch Messaging normally by clicking the Scratch Addons icon in the extension bar.

Если ты хочешь включить ещё больше функций Scratch Addons, зайдите в его страницу настроек, кликнув на его иконку и кликнув на иконку настроек.
