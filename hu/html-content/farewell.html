Búcsúzunk!

A Scratch Addons eltávolítása megtörtént. Köszönjük, hogy használta a bővítményt!

You may reinstall the extension by clicking the install button above. You can also leave a review on the Chrome extension page, the Firefox addon page or the Microsoft Edge addon page.

Az ön által megadott eltávolítási indokokon alapozva itt van néhány egyéb dolog amiket még csinálhat.

  • Problémákba ütközött? Kérjük tudassa velünk itt, vagy készítsen egy GitHub problémátitt.
  • Hiányzik egy jellemző? Mondja el nekünk itt, vagy javasoljon egy probléma készítése által, vagy posztoljon a vitalapunkra.
  • Csalódott, vagy van valami mondanivalója? Ismertesse meg velünk is itt.
